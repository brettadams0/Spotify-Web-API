# Spotify-Web-API
## Overview
Spotify Stats is a web application that allows users to view their Spotify listening statistics. The application uses the Spotify Web API to fetch data related to the user‚Äôs top artists, tracks, genres, and playlists. The application is built using Flask, a Python web framework, and Spotipy, a lightweight Python library for the Spotify Web API.

## Features
User Profile: Displays the user‚Äôs Spotify profile information.
Top Artists: Shows the user‚Äôs top artists based on their listening history.
Top Tracks: Shows the user‚Äôs top tracks based on their listening history.
Top Genres: Displays the most common genres among the user‚Äôs top artists.
Playlists: Lists the user‚Äôs Spotify playlists.

## Installation
Clone this repository.
Install the required Python packages: Flask and spotipy.
Replace 'your_secret_key' in app.secret_key with your own secret key.
Replace SPOTIPY_CLIENT_ID, SPOTIPY_CLIENT_SECRET, and SPOTIPY_REDIRECT_URI with your Spotify Web API credentials.
Run the application using the command python app.py.
Usage
Open your web browser and navigate to http://localhost:5000.
Click on the ‚ÄúLog in with Spotify‚Äù button.
You will be redirected to the Spotify login page. Log in with your Spotify credentials.
After successful login, you will be redirected back to the application where you can view your Spotify stats.
